THE MEME FACTORY 
website preview: ![Project Demo](./assets/screenshot.png)
🖼️ 
The Meme Factory is a dynamic web application designed for meme enthusiasts to discover, explore, and preview popular meme templates. Users can browse a vast library of templates, filter by categories, and visualize their own captions in real-time.

🚀 Purpose
The goal of this project is to provide a seamless, visually engaging interface for interacting with the Imgflip API. It focuses on solving the common UI challenge of displaying varied image aspect ratios while providing interactive tools for content previewing.

🛠️ Core Features
Masonry Grid Layout: A responsive, Pinterest-style layout that handles diverse image dimensions elegantly.

Real-time Meme Creator: A preview tool where users can input text and see it overlaid on their chosen template.

Search & Filter: Find specific templates or browse by popular categories.

Share Functionality: Quick "Copy to Clipboard" feature for meme links.

Smooth Navigation: Includes a "Back to Top" button for long-scroll browsing sessions.

🌐 API Integration
This project utilizes the Imgflip API.

Endpoint: https://api.imgflip.com/get_memes

Usage: Fetches an array of the top 100 most popular meme templates, including names, URLs, and dimensions.

💻 Technologies
Frontend: HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6+)

Styling: Custom CSS for the Masonry effect

Deployment: GitHub Pages / Vercel (Optional)
